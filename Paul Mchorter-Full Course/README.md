# Paul McWhorter - Arduino Practice Code

All my practice code, assignments, and Wokwi circuit simulations from following Paul McWhorter's Arduino tutorial series.

## What's Here
- Lesson code (what Paul solves during videos)
- Homework assignments (what Paul gives students to solve)
- Wokwi circuit animations for every example

## How to Run
1. Open any `.ino` file in Arduino IDE
2. Upload to your Arduino board
3. OR click the Wokwi simulation link to run in browser

Paul McWhorter's Arduino Tutorials for Beginners on YouTube
