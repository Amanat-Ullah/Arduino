# 🔴🟢🟢 Assignment 01 - Multi LED Blinking

## 📌 Objective
Control 3 LEDs (Red, Green, Yellow) using Arduino with different time delays.

---

## 🎯 Task Description
The program blinks three LEDs in sequence:

- 🔴 Red LED → ON for 5 seconds  
- 🟢 Green LED → ON for 10 seconds  
- 🟡 Yellow LED → ON for 15 seconds  

Then the cycle repeats continuously.

---

## 🧠 Concepts Used
- digitalWrite()
- pinMode()
- delay()
- Sequential execution in loop()

---

## ⚙️ Components Used
- Arduino Uno  
- Red LED  
- Green LED  
- Yellow LED  
- 220Ω resistors  
- Breadboard & jumper wires  

---

## 💡 Working Logic
1. Red LED turns ON for 5 seconds  
2. Red turns OFF  
3. Green LED turns ON for 10 seconds  
4. Green turns OFF  
5. Yellow LED turns ON for 15 seconds  
6. Yellow turns OFF  
7. Loop repeats
